Color

En CSS, el color se puede definir de varias maneras:

  • Por nombre: red, blue, etc.
  • Por código hexadecimal: #FF0000 para blue, #0000FF para green, etc.
  • Por valores RGB: rgb(255, 0, 0) para rojo, rgb(0, 0, 255) para azul, etc.
  • Por valores RGBA (con transparencia): rgba(255, 0, 0, 0.5)
  • Por valores HSL: hsl(120, 100%, 50%)
  • Por valores HSLA (con transparencia): hsla(120, 100%, 50%, 0.5)
